# User Stories

1. As a budget-conscious college student, I want to see my dining dollar balance and current bet at all times so that I can easily track my risk and rewards.

2. As a player who likes strategy, I want to choose between different bet amounts so that I can control how risky each spin feels, but make sure there's still a sense of risk on the amount of bets.

3. As a thrill-seeking player, I want winning spins to trigger exciting visual effects so that the game feels rewarding and energetic; like accumulating combos or collecting certain objects to win bigger rewards.

4. As a player in a dorm or library, I want a mute/unmute button so that I can control game audio depending on my environment.

5. As a screen-reader user, I want spin outcomes to be announced through accessible live updates so that I can understand the results without relying only on visuals.

6. As a college student, I want the slot symbols to use student-life themed assets like ramen, textbooks, job application and loose change so that the game feels unique and relatable.

7. As a casual player, I want the controls to be simple and easy to understand so that I can start playing immediately without needing extra explanation (keep the game mechanics basic and subtle effects-not too colorful).
