Below are some notes to consider on existing web based games, before prompting the chosen AI interface.

## 1. Existing Web Based Games
Slot machine games are not a new idea. Therefore, we can take inspiration from existing games to add good features to our game and also be aware so as to avoid copyright issues.

* **Blackbeard Pirate Slot (Playpager version):**HTML5 slot simulator, pirate themed
    * *Implementation Ideas:* Add a fun theme for the slot machine, bonus cannon game attached (to earn more points), control bet size, auto-play option. 
* **Troya Slot:** * Trojan War themed, includes multiplayer/social elements like rankings
    * *Implementation Ideas:* Progression systems, winning combinations unlock mini-games, free spins and bonus rounds, sign in option to save progress
* **Goldilocks VegasSlotsOnline:** Based off Goldilocks and the Three Bears
    * Implementation Ideas:* Sound options, display amount won per spin, themed slot machine, buttons light up when mouse hovers over it, game rules option (about the game, how to play, game functionality description--bet, spin, win, paytable, autoplay, advanced autoplay, fast play, desktop vs mobile version settings)

## 2. Common Expected Features
Here are some features we should make sure to implement and prompt AI to include.
* 3 or 5 reels that spin vertically
* Stops are determined by a random number generator
* Straight or zigzag lines that determine wins
* Each spin is independent and random
* Control bet
* (Optional) control paylines
* Payout system: match symbols with rewards (based on symbol rarity, number of matches, bet size)
* Paytable: explains the rules to the player
* Win animations: highlight winning lines, flashing lights/symbols, exciting sounds

## 3. Optional Additional Features
These are some additional features that could improve user experience. Not all should be added, however, so as to not overwhelm the player.
* Free spins
* Wild symbols: can be used as another symbol
* Bonus rounds/mini games
* Multipliers: usually happen during bonus rounds
* Account login to save progress
